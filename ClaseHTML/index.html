HTML (Lenguaje de Marcado de Hipertexto) es un lenguaje de marcado que estructura el contenido de una página web, mientras que PHP (Hypertext Preprocessor) es un lenguaje de programación del lado del servidor que permite crear sitios web dinámicos e interactivos al generar contenido que se envía al navegador en formato HTML. PHP se ejecuta en el servidor, procesa información y luego crea el HTML que el usuario ve en su navegador, lo que permite funciones como el registro de usuarios, la gestión de carritos de compra y la adaptación del contenido
